R for Data Analysis

Introduction to R for non-programmers.

The goal of this lesson is to teach novice programmers to write code for using R for data analysis. R is commonly used by many data scientists. Surveys show that R is the most popular software for statistical data analysis (76% respondents use R for data analysis). As its popularity grows so does the number of third-party R packages (currently about 7700). This workshop will focus on teaching the fundamentals of the programming language R. However it will not teach statistical analysis.

Note that this lesson is a modified version of the “R for Reproducible Scientific Analysis” lesson from here. It was adapted to fit better the needs of the PSRC Data department.

Prerequisites

Have attended the Shell and Git sessions.

Topics

  1. Introduction to R and RStudio
  2. Data structures
  3. Data frames and reading in data
  4. Subsetting data
  5. Vectorisation
  6. Data Analysis
  7. Creating functions and Control flow
  8. Writing data
  9. Extras
  10. Homework
